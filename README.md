# 🖥 실행 방법

### 1. 패키지 설치

- `yarn` 또는 `yarn install`로 패키지를 설치해줍니다.

### 2. 환경 설정

- `.env` 파일을 루트 디렉토리에 만듭니다.

```
SKIP_PREFLIGHT_CHECK=true
```

- 내용을 입력합니다.

### 3. 실행

- `yarn start` 명령어로 실행합니다.
